# Role-Based Student Record Management System

A console-based Student Record Management System developed in **C** that provides secure authentication and role-based access to manage student information. The application allows administrators, staff members, and guests to perform operations based on their assigned permissions while maintaining persistent storage using text files.

## Features

* User authentication with username and password
* Role-Based Access Control (Admin, Staff, Guest)
* Add, search, update, and delete student records
* Persistent storage using file handling
* Duplicate record prevention
* Permission-based operation restrictions
* Input validation and error handling
* Modular and maintainable code structure

## Technologies

* C
* Structures
* File Handling
* Modular Programming
* Dynamic Memory Management
* Standard C Library
